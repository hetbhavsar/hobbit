loader

样式控件.loading

描述：只要给任意dom元素增加 .loading 样式，它会为该元素动态在元素的中央创建一个 loading 控件。

备注：它会将元素的显示状态变为 inline-block，并且会将该元素设置成relative定位方式，你可以通过 !important 来重写定位方式。 

                    <span class="loading" 
                    style="height:50px;width:80px;">
                    </span>

$H.loader.show({options}) ⇒ $

描述：显示 loader 控件。

参数 类型 必需
text string 否
mask bool 否
scrollable bool 否
clickMask function 否
